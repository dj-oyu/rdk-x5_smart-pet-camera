# プロジェクトゴール - スマートペットカメラ

## プロジェクト概要

AI物体検出技術を活用し、ペット（猫）の日常行動を自動的に記録・分析するスマート監視システムの開発。

## 背景と目的

### 背景
- ペットの健康管理には日々の食事量や水分摂取量の把握が重要
- 飼い主が不在時のペットの行動を記録したいニーズがある
- 手動での記録は手間がかかり、継続が難しい

### 目的
- 猫の水飲み・食事行動を自動的に検出・記録
- 24時間体制での監視（昼夜問わず）
- 行動データの可視化と分析基盤の構築

## 期待される成果

### 主要成果物
1. リアルタイム物体検出システム
2. 行動記録・分析機能
3. 動画・JSONデータの保存システム
4. 行動パターンの可視化ダッシュボード（将来的に）

### ビジネス価値
- ペットの健康状態の早期発見
- 飼い主の安心感向上
- 獣医診断時の客観的データ提供

## プロジェクトスコープ

### 対象範囲
- 猫の食事・水飲み行動の検出と記録
- 昼間カメラ・夜間カメラの2基体制での監視
- 物体検出モデルとバウンディングボックス重なり判定による行動推定
- 記録データの保存（JSON、動画）

### 対象外
- リアルタイムアラート機能（将来的に検討）
- クラウド連携機能（将来的に検討）
- 複数のペット個体識別（将来的に検討）
- スマートフォンアプリ開発（将来的に検討）

## 技術的な前提

### ハードウェア
- カメラモジュール2基（昼間用・夜間用）
- 組み込みLinuxデバイス
- 既存の`capture_v2.c`を基盤とした映像キャプチャシステム

### ソフトウェア
- C言語によるカメラ制御
- Python/C++による物体検出処理
- OpenCVまたは類似のコンピュータビジョンライブラリ
- 軽量な物体検出モデル（YOLO/MobileNet等）

## 成功基準

1. 食事・水飲み行動の検出精度：90%以上
2. 24時間連続稼働の安定性
3. 記録データの正確性と完全性
4. システム応答時間：リアルタイム処理（30fps以上）

## タイムライン

このドキュメントでは具体的な期限は設定せず、以下のフェーズで開発を進める：

1. 基盤整備フェーズ：カメラキャプチャシステムの確立
2. 物体検出フェーズ：検出モデルの統合とチューニング
3. 行動推定フェーズ：バウンディングボックス重なり判定ロジックの実装
4. 記録システムフェーズ：データ保存と管理機能の実装
5. テスト・改善フェーズ：精度向上と安定性確保
