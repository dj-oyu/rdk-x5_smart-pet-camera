# ペットアルバム MCP サーバー

ai-pyramid サーバーに組み込まれた MCP (Model Context Protocol) エンドポイント。
外部の AI エージェント（picoclaw 等）から猫の写真メタデータを取得し、画像をダウンロードできる。

## エンドポイント

| パス | メソッド | プロトコル | 説明 |
|------|---------|-----------|------|
| `/mcp` | POST | MCP Streamable HTTP (JSON-RPC 2.0) | ツール呼び出し |
| `/mcp/photos/{id}` | GET | HTTP | JPEG 画像ダウンロード (DB ID 指定) |

## MCP ツール

### `get_recent_photos`

VLM 分析済みの猫写真メタデータを新しい順に取得する。

**パラメータ:**

| 名前 | 型 | デフォルト | 説明 |
|------|-----|-----------|------|
| `limit` | integer | 20 | 取得件数 (max 50) |
| `pet_id` | string | — | `"chatora"` または `"mike"` でフィルタ |

**レスポンス例:**

```
5 photos (total 847):
#842 | 2026-03-22 14:30 | chatora | resting | "ソファの上で丸くなっている茶トラ猫"
#841 | 2026-03-22 14:15 | mike | playing | "おもちゃのネズミに飛びかかるミケ猫"
#839 | 2026-03-22 13:45 | chatora | eating | "フードボウルからカリカリを食べている"
#837 | 2026-03-22 13:00 | chatora | sleeping | "キャットタワーの最上段で熟睡中"
#835 | 2026-03-22 12:20 | mike | grooming | "窓辺で毛づくろいをしているミケ猫"
```

各行の `#ID` は画像ダウンロードに使う DB プライマリキー。

### 画像ダウンロード

MCP ツールではなく通常の HTTP GET。

```
GET /mcp/photos/842
→ Content-Type: image/jpeg
```

## picoclaw 設定例

`~/.picoclaw/config.json`:

```json
{
  "tools": {
    "mcp": {
      "servers": {
        "pet_album": {
          "enabled": true,
          "type": "http",
          "url": "https://m5stack-ai-pyramid.tail848eb5.ts.net:8090/mcp",
          "deferred": false,
          "timeout": 30
        }
      }
    }
  }
}
```

## プロンプトのアイデア

以下は picoclaw のシステムプロンプトや cron タスクに組み込むことを想定した例。

### 定期レポート（1日のまとめ）

> `get_recent_photos` で今日の写真を取得し、猫たちの1日の様子をまとめてください。
> 何時頃に活動的だったか、食事・睡眠・遊びのパターンに注目して、
> 飼い主が読んで楽しい日報を作ってください。
> 特徴的なシーンがあれば、その写真（1〜3枚）をダウンロードして添付してください。

### 異変検知・プロアクティブ通知

> 定期的に `get_recent_photos(limit=5)` をポーリングし、以下に該当する場合は通知してください：
>
> - 長時間（4時間以上）写真が撮れていない（カメラ異常 or 猫が不在）
> - 同じ猫が長時間 "resting" や "sleeping" のまま姿勢が変わらない
> - いつもと違う時間帯に "eating" が記録された
> - "is_valid=false" の写真が連続している（VLM異常 or カメラの画角ずれ）
>
> 通知は簡潔に、心配しすぎないトーンで。

### 猫の生活リズム分析

> 過去の写真メタデータから、chatora と mike それぞれの生活リズムを分析してください。
>
> - 活動時間帯のピーク
> - 食事の頻度とタイミング
> - 睡眠が多い時間帯
> - 二匹が同時に写っている時間帯（仲良し度の指標）
>
> 結果をわかりやすくまとめて、気づいたことがあればコメントしてください。

### 写真セレクション（ベストショット）

> 最近の写真から、飼い主が喜びそうなベストショットを3枚選んでください。
> 選ぶ基準：
> - キャプションが面白い・かわいい描写のもの
> - 珍しい行動（grooming, playing）が写っているもの
> - 二匹のやりとりが想像できるもの
>
> 選んだ写真をダウンロードして、一言コメントをつけて送ってください。

### 来客時のお留守番モニター

> 今から外出します。30分ごとに最新の写真をチェックして、
> 猫たちが落ち着いているか確認してください。
> 問題なければ「大丈夫」の一言でOK。
> 様子がおかしければ写真付きで教えてください。
